# Тренажер слов

## Описание
Приложение-тренажер для изучения и запоминания иностранных слов. Позволяет пользователям самостоятельно добавлять нужные слова, которые они в дальнейшем смогу выучить.

## Основные возможности
- Добавление новых слов с переводом
- Режим изучения с карточками
- Отслеживание прогресса обучения
- Удаление ненужных слов
